Connect to GitHub

The admin reads and saves content directly to your GitHub repo. Enter your GitHub Personal Access Token once — it stays in your browser only.

How to get your token (takes 2 minutes):
1. Go to github.com/settings/tokens/new
2. Note: "Evalia Admin" · Expiration: No expiration
3. Tick repo (full repository access)
4. Click Generate token → copy and paste above
