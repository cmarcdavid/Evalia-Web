Connect to GitHub

The admin reads and saves all content directly to your GitHub repo. Enter your Personal Access Token once — it's stored only in your browser.

Get your token in 2 min:
1. Go to github.com/settings/tokens/new
2. Note: "Evalia Admin" · Expiry: No expiration
3. Tick repo (full access)
4. Generate → copy → paste above
