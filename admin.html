Connect to GitHub

Enter your GitHub Personal Access Token once — stored only in your browser.

Get token (2 min):
1. github.com/settings/tokens/new
2. Note: "Evalia Admin" · No expiration · Tick repo
3. Generate → copy → paste above

⚠️ Rate limited? GitHub limits API calls to 60/hr without a token.
If you see "rate limit" errors, wait a few minutes then try again — your token is fine.

Guest Detail
Integration Ready: QuickBooks / Zoho / Stripe
Guest records are structured for direct integration with billing systems.
QuickBooks: Export CSV → Import as Customers (Name, Email, Phone fields map directly).
Zoho CRM: Use Export CSV → Contacts import. Or use Zoho's GitHub integration to sync _data/guests/ automatically.
Stripe: Use the exported email list to create Stripe customers via the Stripe dashboard or API.
WhatsApp Business API: Phone numbers are stored in international format, ready for WhatsApp Business messaging campaigns.
