EVALIA WEBSITE FILES
====================

Files included:
  index.html          → Public website (drag into your GitHub repo folder)
  admin.html          → Admin dashboard
  ARCHITECTURE.md     → Technical documentation
  SETUP_GUIDE.md      → Formspree + Cloudinary setup guide
  env.example.txt     → Rename to: .env.example  (before adding to GitHub)
  gitignore.txt       → Rename to: .gitignore    (before adding to GitHub)

IMPORTANT — Before pushing to GitHub, rename:
  env.example.txt  →  .env.example
  gitignore.txt    →  .gitignore

To rename on Mac:
  1. Click the file once to select it
  2. Press Enter to rename
  3. Type the new name with the dot at the start
  4. Press Enter to confirm

Note: Files starting with a dot are hidden in Finder by default.
To show hidden files: press Cmd + Shift + . (dot) in any Finder window.
